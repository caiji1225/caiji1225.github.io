zzcms2019+phpstudy

1、环境搭建

windows：

​ phpstudy

​ IIS|Apache|Nginx + PHP +MySQL

Linux|Unix：

​ Apache + PHP + MySQL （PHP须在非安全模式下运行）

zzcms 目录结构

1.1网站搭建步骤

1、上传程序源码到网站服务器根目录 （源码目录D:\phpstudy\PHPTutorial\WWW\zzcms2019）

2、打开phpstudy 其他选项菜单=》站点域名管理

添加网站域名 自定义：www.zzcms.com

网站目录 D:\phpStudy\PHPTutorial\WWW\zzcms2019\

3、打开host 文件

添加：127.0.0.1 www.zzcms.com

4、浏览器访问 www.zzcms.com/install 按照步骤安装 就ok。

5、按照要求安装程序，此时网站已经成功运行( 安装完删除install 目录以及里面的文件，)

PHP环境和系统函数

1.2 PHPStorm配置

1、默认安装PHPStorm后哦，打开。
2、file=》settings=》Languages & Frameworks =>PHP
此时配置PHP环境
1、找到 CLI Interpreters 下存在
PHP executable：D:\phpStudy\PHPTutorial\php\php-5.6.27-nts\php.exe（PHP的绝对路径）
Debugger extension：D:\phpStudy\PHPTutorial\php\php-5.6.27-nts\ext\php_xdebug.dll （debug的绝对路径）
Apply=》OK
2、配置PHP.ini
phpstudy页面切换对应版本，然后=》其他选项菜单=》打开配置文件=》PHP.ini （此时打开的ini文件是对应php版本的文件）

php.ini配置完整说明

php.ini内容修改编辑

3、配置PHPStorm 的Debug端口9003
    在file—>settings—>languages&frameworks—PHP—>debug =》Xdebug =>Debug port 设置端口为9003(和php.in中的保持一致
4、在file—>settings—>languages&frameworks—PHP—>debug=》DBGp Proxy
    IDE key：PHPSTORM	（PHP.ini 中的xdebug.idekey=PHPSTORM）
    host：localhost
    port：9003
5、在file—>settings—>languages&frameworks—PHP=》servers 
    Name：PHP
    Host：localhost		Port：80		Debugger：Xdebug
6、配置运行时选项
    Run/Debug Configuration=>PHP Web Page
        Server :PHP （自己之前定义的name）
        start URL：http://www.zzcms.com
Apply=>OK

此时整个环境已经配置完成。
