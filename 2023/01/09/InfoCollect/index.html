5、端口信息收集
1、nmap
nmap -A -v -T4 -O -sV 目标地址
nmap [ <扫描类型> …] [ <选项> ] { <扫描目标说明> }j
基础命令：
-A 全面扫描
-sP Ping 扫描 - 只需确定主机是否在线
-v 显示扫描过程
-O 探测主机操作系统
-p 指定端口扫描
-sS (TCP SYN 扫描)
-sT (TCP connect() 扫描)
-sU (UDP 扫描)
-sN; -sF; -sX (TCP Null，FIN，and Xmas 扫描)
-Pn (不通过 ICMP 探测)
基础扫描：
全面进攻性扫描（包括各种主机发现、端口扫描、版本扫描、OS扫描及默认脚本扫描）:
nmap -A -v target_ip
ping扫描: nmap -sn -v target_ip
快速端口扫描: nmap -F -v target_ip
版本扫描: nmap -sV -v target_ip
操作系统扫描: nmap -O -v target_ip
存活扫描：nmap -sn x.x.x.x/24
全端口识别：nmap -sV -p 1-65535 x.x.x.x

8、社会工程学
研究：信息收集（WEB、媒体、垃圾桶、物理），确定并研究目标
钩子：与目标建立第一次交谈（HOOK、下套）
下手：与目标建立信任并获取信息
退场：不引起目标怀疑的离开攻击现场

常见信息包括：姓名、性别、出生日期、身份zheng号、身份zheng家庭住址、身份zheng所在公安局、快递收货地址、大致活动范围、qq、手机号、邮箱、银行card号（银行开户行）、支付宝、贴吧、百度、微博、猎聘、58、同城、网盘、微信、常用ID、学历（小/初/高/大学/履历）、目标性格详细分析、常用密码、照片EXIF信息。
常见可获取信息系统包括：中航信系统、春秋航空系统、12306系统、三大运营商网站、全国人口基本信息资源库、全国机动车/驾驶人信息资源库、各大快递系统（越权）、全国出入境人员资源库、全国在逃人员信息资源库、企业相关系统、全国安全重点单位信息资源库等。
