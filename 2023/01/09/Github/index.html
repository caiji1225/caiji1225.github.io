1、资源管理器 （使用Github提供的GraphQL来查询用户邮箱）
https://docs.github.com/cn/graphql/overview/explorer
{
repository(name: “grank”, owner: “lctt”) {
ref(qualifiedName: “master”) {
target {
… on Commit {
id
history(first: 5) {
edges {
node {
author {
name
email
}
}
}
}
}
}
}
}
}
并将 name 替换为你要查询的人的 repo 名，owner 改为你需要查询的人的名字，然后点击执行按钮。
一种是我们常见的，自己用的各种免费邮箱，比如 @qq.com、@gmail.com、@foxmail.com 之类的；
另一种是形如 27856297+dependabot-preview[bot]@users.noreply.github.com 这样的邮箱
前者是我们自己通过 git 设置的邮箱，而后者则是我们通过 Github 网页、 API 操作产生的 commit
你在查询的时候，要记得去找第一类邮箱来作为参考
